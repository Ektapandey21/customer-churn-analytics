# Customer Segmentation & Churn Analytics

## Project Overview
This project analyzes customer churn in European banking using segmentation techniques.

## Objectives
- Identify high-risk churn segments
- Analyze churn across geography, age, and financial attributes
- Build an interactive dashboard

## Files Included
- Jupyter Notebook (EDA & Analysis)
- Streamlit Dashboard (app.py)
- Dataset
- Project Report
